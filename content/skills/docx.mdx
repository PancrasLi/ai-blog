# Word Document Creator

## Overview

Word Document Creator Skill 提供完整的 .docx 文档生成和编辑能力，支持专业格式化和版本控制。

## Key Features

### 文档操作
- 新建文档
- 编辑现有文档
- 保留格式化设置
- 支持表格和图表

### 高级特性
- 修订跟踪
- 批注管理
- 文档模板
- 内容提取

## 在 Claude Code 中引入

```typescript
import { WordDocument } from 'openclaw-skills/docx';

const doc = new WordDocument();
doc.addParagraph('标题', { fontSize: 14, bold: true });
doc.addParagraph('正文内容');
await doc.save('document.docx');
```

## 在 Cursor 中引入

```
#docx create 文档标题
#docx edit document.docx
```

## 在 OpenClaw 中引入

```yaml
skills:
  - name: docx
    enabled: true
    config:
      preserveFormatting: true
      enableTracking: true
```
