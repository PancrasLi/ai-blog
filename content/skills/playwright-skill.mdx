## 介绍

**playwright-skill** 是一个强大的浏览器自动化 Skill，基于 Playwright 框架，允许 Claude Code 自主编写和执行浏览器自动化脚本。无需手工干预，AI 可直接控制浏览器进行自动化测试、数据抓取和交互验证。

### 核心功能
- 浏览器自动化脚本编写和执行
- 跨浏览器测试（Chromium, Firefox, WebKit）
- 网页元素识别和交互（点击、输入、滚动）
- 自动化表单填充和提交
- 网页截图和录屏
- 性能监控和网络请求拦截
- 模型自主调用（Model-invoked）

## 使用方式

### 在 Claude Code 中引入

Claude Code 是 Claude 的代码执行环境，原生支持 playwright-skill：

1. **安装依赖**
```bash
npm install -D @playwright/test
npm install playwright
```

2. **在代码中导入和使用**
```javascript
// 导入 Playwright
const { chromium } = require('playwright');

async function testWebsite() {
  const browser = await chromium.launch();
  const page = await browser.newPage();

  // 访问网站
  await page.goto('https://example.com');

  // 点击元素
  await page.click('button#submit');

  // 获取文本内容
  const title = await page.title();
  console.log(`页面标题: ${title}`);

  // 截图
  await page.screenshot({ path: 'screenshot.png' });

  await browser.close();
}

testWebsite();
```

3. **在 Claude Code 编辑器中执行**
- 复制上述代码到 Claude Code
- Claude 将自动执行并返回结果
- 支持交互式调试和错误恢复

## 完整示例

### 示例 1：自动化登录和数据抓取

```javascript
const { chromium } = require('playwright');

async function scrapeAfterLogin() {
  const browser = await chromium.launch();
  const page = await browser.newPage();

  try {
    // 导航到登录页
    await page.goto('https://api.example.com/app');

    // 登录
    await page.fill('input[name="username"]', 'admin');
    await page.fill('input[name="password"]', 'securepass');
    await page.click('button[type="submit"]');

    // 等待页面加载
    await page.waitForLoadState('networkidle');

    // 抓取数据
    const data = await page.evaluate(() => {
      return Array.from(document.querySelectorAll('tr')).map(row => ({
        name: row.querySelector('td:nth-child(1)')?.textContent,
        status: row.querySelector('td:nth-child(2)')?.textContent,
      }));
    });

    console.log('抓取数据:', data);

    // 生成报告
    await page.screenshot({ path: 'report.png' });

  } finally {
    await browser.close();
  }
}

scrapeAfterLogin();
```

### 示例 2：性能监控和网络请求拦截

```javascript
const { chromium } = require('playwright');

async function monitorPerformance() {
  const browser = await chromium.launch();
  const page = await browser.newPage();

  // 记录网络请求
  const requests = [];
  page.on('request', request => {
    requests.push({
      url: request.url(),
      method: request.method(),
      timestamp: new Date(),
    });
  });

  // 测量性能
  const startTime = Date.now();
  await page.goto('https://example.com');
  const loadTime = Date.now() - startTime;

  console.log(`页面加载时间: ${loadTime}ms`);
  console.log(`网络请求数: ${requests.length}`);

  await browser.close();
}

monitorPerformance();
```

## 常见问题

**Q: 支持哪些浏览器？**
A: 支持 Chromium, Firefox, WebKit，可同时运行多个浏览器实例

**Q: 如何处理动态加载的内容？**
A: 使用 waitForLoadState() 或 waitForSelector() 等待元素出现

**Q: 可以录制用户行为吗？**
A: 支持录屏功能，使用 page.video.path() 自动录制浏览器操作

**Q: 如何提高性能？**
A: 使用无头模式、禁用图片加载、并行执行多个浏览器实例

**Q: 支持 JavaScript 执行吗？**
A: 完全支持，可直接在浏览器上下文中执行任意 JS 代码
