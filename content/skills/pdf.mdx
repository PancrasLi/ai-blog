## 介绍

**pdf** Skill 提供了完整的 PDF 处理能力，支持文本提取、表格识别、文档合并/拆分、表单填充等功能。

### 核心功能
- 从 PDF 中提取文本和表格
- 创建新的 PDF 文档
- 合并和拆分 PDF 文件
- 填充 PDF 表单
- 添加水印和注释
- 管理 PDF 元数据

## 使用方式

### 在 Claude Code 中引入

```javascript
import { PDFHandler } from '@/skills/pdf';

// 从 PDF 中提取文本
const pdfHandler = new PDFHandler();
const text = await pdfHandler.extractText('document.pdf');

// 创建新 PDF
await pdfHandler.createPDF('output.pdf', {
  title: '生成的文档',
  content: '这是自动生成的 PDF'
});
```

### 在 Cursor 中引入

1. 打开 Terminal，安装依赖：
```bash
npm install pdfjs-dist pdf-lib
```

2. 在代码中使用：
```typescript
import * as PDFLib from 'pdf-lib';

const pdfDoc = await PDFLib.PDFDocument.create();
const page = pdfDoc.addPage([595, 842]); // A4 页面
const pdfBytes = await pdfDoc.save();
```

## 常见问题

**Q: 支持哪些文件格式？**
A: 主要支持 PDF 格式，可以处理扫描件和数字化 PDF

**Q: 可以进行 OCR 识别吗？**
A: 支持基础的文本识别，但对于复杂的手写内容，建议使用 OCR 专门工具

**Q: 如何处理加密的 PDF？**
A: 需要先解密，然后才能进行处理
