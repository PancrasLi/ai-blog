# Self-Improving Agent

## Overview

Self-Improving Agent Skill 是实现 AI 智能体长期学习和进化的关键，通过持续记录和反馈优化，让 Agent 性能不断提升。

## Key Features

- 错误记录
- 学习捕获
- 反馈整合
- 持续优化

## 在 Claude Code 中引入

```typescript
import { SelfImprovingAgent } from 'openclaw-skills/self-improving-agent';

const agent = new SelfImprovingAgent();

agent.recordLearning('learning_key', insight);
```
