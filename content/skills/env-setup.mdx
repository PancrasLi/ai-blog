# Environment Auto-Setup

## Overview

Environment Auto-Setup Skill 自动化环境配置流程，智能检测和安装所需依赖，节省开发时间。

## Key Features

- 自动依赖检测
- 跨平台支持
- 一键安装
- 配置验证

## 在 Claude Code 中引入

```typescript
import { EnvSetup } from 'openclaw-skills/env-setup';

const setup = new EnvSetup();
await setup.detectMissing();
await setup.installDependencies();
```

## 在 Cursor 中引入

```
#setup auto-detect
#setup install
```

## 在 OpenClaw 中引入

```yaml
skills:
  - name: env-setup
    enabled: true
    platforms:
      - macos
      - linux
```

## 支持的工具

- Node.js 和 npm 包管理器
- Python 和 pip
- Git
- Docker
- 开发工具链

## 工作原理

1. 扫描系统环境
2. 检测缺失工具
3. 自动安装依赖
4. 验证安装成功
