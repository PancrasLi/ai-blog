# Presentation Builder

## Overview

Presentation Builder Skill 让你能够以编程方式创建和编辑专业的 PowerPoint 演示文稿。

## Key Features

- 幻灯片管理
- 布局选择
- 演讲者备注
- 动态内容插入

## 在 Claude Code 中引入

```typescript
import { Presentation } from 'openclaw-skills/pptx';

const ppt = new Presentation();
const slide = ppt.addSlide('Title Slide');
slide.addTitle('演示标题');
await ppt.save('presentation.pptx');
```

## 在 Cursor 中引入

```
#pptx create 演示主题
#pptx add-slide
```
